## Libs description

- Wire (I2C)
  - Use PIN:
    - SDA: Data line;
    - SCL: Clock line;
- SPI (Serial Peripheral Interface)
  - Use PINs:
    - MISO: Master In Slave Out;
    - MOSI: Master Out Slave In;
    - SCK: Serial Clock;
    - SS: Slave Select.
- OneWire
  - Use PIN:
    - Any Digital PIN
